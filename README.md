# Catálogo Mobile

Aplicativo mobile desenvolvido com React Native e Expo.

## Funcionalidades

- Login com validação
- Listagem de produtos por categoria
- Consumo de API com Axios
- Tabs Masculino / Feminino
- Tela de detalhes do produto
- Botão de compra
- Logout

## Tecnologias

- React Native
- Expo
- Axios
- React Navigation

## Como executar

```bash
npm install
npx expo start
